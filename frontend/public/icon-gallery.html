Usage Notes:
  • Gear slot icons are stored locally in public/images/gear-slots/{variant}/
  • Current active variant: white with CSS brightness-[3.0] filter
  • To change variant, update GEAR_SLOT_ICONS in types/index.ts
  • Job icons and material icons are loaded from XIVAPI CDN
  • Viper and Pictomancer use numeric icon format: /i/062000/0620XX.png
  • Other jobs use classic format: /cj/1/{jobname}.png (lowercase)
XIVAPI Icon URLs:
  • Surgelight Twine: https://xivapi.com/i/021000/021686.png
  • Surgelight Glaze: https://xivapi.com/i/027000/027636.png
  • Surgelight Solvent: https://xivapi.com/i/027000/027637.png
  • Universal Tomestone: https://xivapi.com/i/026000/026652.png
Regenerate Gear Icons: 
cd frontend && python scripts/colorize-gear-icons.py

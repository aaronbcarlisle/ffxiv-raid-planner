# Plan C — Generalized Settings Panel + Role-Aware Tabs + Site-Wide Tab Persistence

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn the static-only "Static Settings" panel into a role-aware **Settings** panel (tabs filtered by role; Members get read-only Priority/Members), rename the static config tab General → **Static**, add a new **General** tab holding *user* settings, and add a synced user preference that controls whether navigational tabs are **remembered** or **reset to default** site-wide.

**Architecture:** Add a `tab_persistence` column to the `User` model + `/me/preferences` endpoint (mirrors `activity_display_mode`). Frontend reads it via `authStore`. A small `lib/tabMemory.ts` gates all navigational tab persistence on that preference, keyed per-static by the `:shareCode` route param. The Settings panel filters tabs by role and renders a new General tab whose first control is the remember/reset toggle.

**Tech Stack:** FastAPI + SQLAlchemy + Alembic + Pydantic (backend); React 19 + TS + Zustand + Tailwind v4 (frontend); pytest + Vitest/@testing-library.

**Files:**

**Interfaces:**

```ts
// frontend/src/lib/tabMemory.ts
import { useAuthStore } from '../stores/authStore';

function mode(): 'remember' | 'reset' {
  return useAuthStore.getState().user?.tabPersistence ?? 'remember';
}

export function tabKey(base: string, scope?: string): string {
  return scope ? `${base}:${scope}` : base;
}

export function recallTab<T extends string>(key: string, valid: readonly T[], fallback: T): T {
  if (mode() === 'reset') return fallback;
  try {
    const raw = localStorage.getItem(key);
    if (raw && (valid as readonly string[]).includes(raw)) return raw as T;
  } catch { /* ignore */ }
  return fallback;
}

export function rememberTab(key: string, value: string): void {
  if (mode() === 'reset') return;
  try { localStorage.setItem(key, value); } catch { /* ignore */ }
}
```

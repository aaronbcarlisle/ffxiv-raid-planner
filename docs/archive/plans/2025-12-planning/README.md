# Historical Planning Documentation

**Archive Date:** January 1, 2026

These files represent the planning phase of the FFXIV Raid Planner project from December 2025 - January 2026. They are preserved for historical reference.

## Archived Files

- **ARCHITECTURE_SPEC.md** - Initial architecture and integration specification (pre-implementation)
- **ROADMAP.md** - Original feature roadmap (Phases 1-8)
- **LOOT_PRIORITY_UPDATES_PLAN.md** - Detailed loot system implementation plan (90% implemented)
- **LOOT_SYSTEM_AUDIT_IMPLEMENTATION.md** - Loot system implementation checklist

## Status

Most features described in these documents have been implemented. For current project status and outstanding tasks, see:

**Current Documentation:** `/docs/CONSOLIDATED_STATUS.md`

## Why Archived?

These planning documents served their purpose but became outdated as:
- Features were implemented differently than planned
- Requirements changed during development
- Multiple overlapping plans created confusion

The consolidated status document now provides a single source of truth for what's completed, what's outstanding, and what's planned.

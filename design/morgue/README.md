# Design Morgue

This directory contains archived UI concept mockups and design iterations from early development.

These files are **not used** in the current codebase but are preserved for historical reference and potential future inspiration.

## Contents

- `concept-*.jsx` - Various UI layout concepts explored during initial design phase
- `header-redesign-mockup.jsx` - Header redesign exploration
- `priority-panel-redesign.jsx` - Priority panel layout concepts

## Note

These are static mockups, not functional React components. They served as visual references during design discussions.
